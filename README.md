## How to Use
<br>

1. Clone the repository:

```bash
git clone https://github.com/KD-Long/Earth.git
```

2. Install the dependencies:

```bash
cd Earth
npm install
```

3. Run the project:

```bash
npm run dev
```

3. Open your web browser and navigate to http://localhost:5173 to access the project.
